# Политика конфиденциальности CelebrationAI

**Дата вступления в силу: 8 марта 2026 г.**

Мы ответственно относимся к вашим данным. В данной политике описано, как приложение CelebrationAI собирает, использует и защищает информацию.

## 1. Сбор и использование данных

### 1.1. Контакты
Приложение может запрашивать доступ к вашим **Контактам** исключительно для функции импорта дней рождения. Данные контактов обрабатываются локально на вашем устройстве и не передаются на наши серверы.

### 1.2. Firebase и Google Auth
Мы используем **Firebase Authentication** для входа в систему. При использовании Google Sign-In мы получаем ваш публичный профиль (имя, email, фото профиля). Эти данные используются только для синхронизации ваших праздников между вашими устройствами.

### 1.3. Искусственный интеллект (AI)
Для генерации поздравлений и фактов используются нейросетевые модели через API OpenRouter. При отправке запроса на генерацию передаются только те данные, которые вы указали для праздника (имя, дата, интересы), чтобы сделать поздравление персонализированным. Эти данные не используются для обучения глобальных моделей.

## 2. Хранение данных
Ваши данные о праздниках хранятся:
*   Локально на вашем устройстве в базе данных Room.
*   В защищённом облачном хранилище Firebase (если вы вошли в аккаунт), чтобы вы не потеряли их при смене устройства.

## 3. Разрешения (Permissions)
Приложение запрашивает:
*   **READ_CONTACTS**: Для импорта дат из телефонной книги.
*   **INTERNET**: Для работы синхронизации и функций AI.
*   **POST_NOTIFICATIONS**: Для отправки напоминаний о предстоящих событиях.
*   **USE_BIOMETRIC**: Для защиты доступа к приложению.

## 4. Удаление данных
Вы можете в любой момент удалить свои данные прямо в приложении или выйти из аккаунта. При удалении аккаунта все синхронизированные данные удаляются из базы Firebase безвозвратно.
