# Roadmap проекта CelebrationAI 🚀

Список идей и функций для будущих обновлений.

## 📍 Текущие задачи (Выполнено)
- [x] **Информационная сводка (i):** Краткая карточка с интересными фактами (Знак зодиака, восточный календарь, знаменитости-тезки).
- [x] **Гибкие уведомления:** Настройка напоминаний за 0-21 день (слайдер). Индивидуально для каждого и глобально в настройках.
- [x] **Поддержка буквы Ё:** Все генерации ИИ и интерфейс теперь используют правильное правописание.
- [x] **Оптимизация AI:** Использование `openrouter/auto` для быстрой и стабильной генерации (5-10 сек).
- [x] **Черный юмор:** Режим "прожарки" (roast) — жестко, цинично и смешно.
- [x] **Визуальный отклик:** Кнопки меняют цвет при наличии данных (подарок упакован, инфо собрана).

## 🎨 Визуализация и Мультимедиа
- [ ] **AI-генерация открыток:** Создание уникальных изображений на основе описания человека.
- [x] **Анимации:** Добавление праздничных эффектов (конфетти) при открытии карточки именинника.

## 🤖 Улучшение AI-функций
- [x] **Базовая генерация:** Стабильная работа поздравлений и идей подарков.
- [x] **"Извинительные" поздравления:** AI-генерация для тех, кто забыл поздравить вовремя.
- [x] **Многоязычность:** Генерация поздравлений на выбранном языке (RU/EN).

## 📲 Интеграция и Автоматизация
- [x] **Импорт контактов:** Автоматический поиск дней рождения в телефонной книге.
- [x] **Базовая облачная синхронизация:** Сохранение данных через Firebase Firestore.
- [x] **Вишлисты (Wishlists):** Хранение ссылок на подарки прямо в карточке.

## 🔐 Технические улучшения
- [x] **Биометрическая защита:** Вход по отпечатку пальца или FaceID.
- [x] **Интерактивный виджет:** Список праздников на рабочем столе.
- [x] **Архив поздравлений:** Сворачиваемая история поздравлений по годам.
